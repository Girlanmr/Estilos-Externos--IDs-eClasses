Estilos Externos, IDs e Classes

Com o CSS externo, pode-se linkar as páginas da internet com um arquivo .css externo, que se pode criar usando qualquer editor de texto no dispositivo (ex.: Notepad++). Este tipo de CSS é um método mais eficiente, especialmente se está sendo estilizando um site grande. Ao editar um arquivo .css, pode-se modificar um site inteiro de uma só vez.

Adicionando o comando link no red junto de 'href='csss/style.css' type='text/cssq rel='stylesheet', depois aperta Ctrl+N para criar um novo arquivo, cria-se uma pasta chamada css e então salva o novo arquivo criado como style.css, a tag link serve pra chamar arquivos de fora, o href vai ser o endereço por isso cria-se uma pasta chamada css e em seguida coloca-se o nome do arquivo, o type será o tipo que no caso é texto e o rel é para dizer que será um arquivo de estilo, então todo o estilo do site será feito na nova aba do arquivo style.css.
